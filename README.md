## windows build

to package this app for windows:

1. copy the project to a windows machine
2. install python for windows
3. double click `build_windows.bat` or run it in cmd
4. get the packaged app from `dist\AnimeDownloader.exe`

note:

- the packaged app stores its settings in `%LOCALAPPDATA%\AnimeDownloader`
- `ffmpeg.exe` still needs to be installed and available in `PATH` for video downloads
